[Latent Scope](https://github.com/ayushgharat/Latent-Scope) is where I go deeper on the question that drives most of my work: what's actually happening inside a neural network? I trained a small transformer on children's stories and used a Sparse Autoencoder to decompose its hidden states into 2,048 interpretable features — grammatical patterns, semantic concepts, and structures we don't have names for yet. The [web interface](https://latent-scope.vercel.app) lets you type any sentence and watch which internal concepts activate in real time. The entire pipeline, transformer training, SAE training, feature labeling with GPT-4, ran on my MacBook M3. Built on ideas from Anthropic's work on monosemanticity and superposition.

[ViewSense](https://github.com/ayushgharat/viewsense) is a visual diary for blind and visually impaired users. It runs a Vision Language Model and persistent vector store entirely in the browser, zero backend, fully offline, no data ever leaves your device. Forgot where you left something? Ask ViewSense. Want to know what your surroundings looked like five minutes ago? It knows. Built in a weekend at the Man vs. Machine hackathon with no AI coding assistants, placed 2nd.
